### 项目分析（头脑风暴）
####  N (Need)需求
+ 方便便捷
+ 休闲娱乐
+ 创新玩法


#### A(Approach)做法
+ 融合俄罗斯方块与中国传统象棋
+ 利用中国象棋的规则进行方块消除
+ 采用计时规则
+ 提供基础版进阶版不同难度梯度
+ 限时双人对战
+ 限时决策

#### B(Benefit)好处
+ 宣扬中国传统文化象棋
+ 碎片化时间休闲益智
+ 吸引受众了解象棋规则

#### C(Competitors) 竞争
目前存在很多象棋游戏或者俄罗斯方块游戏，但是没有将二者进行结合。本软件有如下竞争力：
+ 创新性高
+ 融入中国传统文化象棋


#### D(Delivery) 推广
+ 发行到微信小程序平台
+ 朋友圈推广
+ 高校圈分享
